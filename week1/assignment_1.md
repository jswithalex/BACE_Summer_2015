## Some Exercises

var names = ['bob','mary','alice','derek','jane'];

PROBLEM 1: 

While names has names in it, call names.pop() and print the resulting names array.

PROBLEM 2: 

To convert celcius to fahrenheit, we multiply C by 1.8 and add 32.  Write a function called `c_to_f` that takes in a Celcius value and returns a Fahrenheit value.

PROBLEM 3: 

Write the inverse of Problem 2 and call it f_to_c.
